# Stock Diary (주식 가계부) 📈

단일 HTML 파일로 동작하는 스마트한 개인 주식 매매 내역 관리 및 AI 상담 웹 애플리케이션입니다.

## ✨ 주요 기능

- **매매 기록 관리**: 직관적인 UI로 주식 매수 내역을 기록하고 수정/삭제할 수 있습니다.
- **포트폴리오 분석**: 보유 종목 수, 총 투자금액, 업종별 비중을 자동으로 계산하여 보여줍니다.
- **데이터 시각화**: Chart.js를 사용하여 업종별 투자 비중을 도넛 차트로 시각화합니다.
- **AI 투자 상담**: Google Gemini API를 연동하여 내 포트폴리오를 기반으로 한 투자 조언을 받을 수 있습니다.
- **데이터 백업/복원**: CSV 및 JSON 파일 형식으로 데이터를 내보내거나 가져올 수 있습니다. 모든 데이터는 브라우저의 `localStorage`에 저장됩니다.

## 🚀 시작하기

이 프로젝트는 단일 HTML 파일로 구성되어 있어 별도의 설치가 필요하지 않습니다.

1. `stock-diary.html` 파일을 다운로드합니다.
2. 브라우저(Chrome, Edge 등)에서 파일을 엽니다.
3. **설정** 탭에서 [Google AI Studio](https://aistudio.google.com/app/apikey)에서 발급받은 Gemini API Key를 입력하면 AI 상담 기능을 사용할 수 있습니다.

## 🛡️ 데이터 보안 안내

- 본 애플리케이션은 서버를 사용하지 않습니다.
- 입력한 모든 데이터와 API Key는 사용자의 브라우저(`localStorage`)에만 저장되며, 외부로 전송되지 않습니다 (Gemini API 호출 시 포트폴리오 데이터 제외).
